# 9.3 精细结构常数的宇宙学漂移

在第六章中，我们论证了耦合常数（如精细结构常数 $\alpha \approx 1/137$）是由 QCA 网络的拓扑连通度决定的。

$$\alpha \sim \frac{\text{内部通道}}{\text{外部通道}}$$

然而，宇宙正在膨胀。在 QCA 视角下，膨胀不仅仅是格点间距拉大（那是度规效应），可能还伴随着**网络拓扑结构的演化**（例如，格点数的增加或连接方式的重组）。

如果宇宙早期的网络连接度（Connectivity）与现在略有不同，那么 $\alpha$ 就不应该是一个绝对常数，而是一个随宇宙时间 $t$ 缓慢变化的标量场。

## 9.3.1 观测方案

通过观测遥远类星体（Quasars）的光谱。这些光来自于百亿年前。我们可以测量其中的原子吸收线（如铁或镁的精细分裂），从而推算出当时的 $\alpha$ 值。

## 9.3.2 预言

$$\frac{\Delta \alpha}{\alpha} = \frac{\alpha(z) - \alpha(0)}{\alpha(0)} \neq 0$$

我们预期 $\alpha$ 存在极微小的宇宙学漂移（例如 $10^{-6}$ 量级）。韦伯望远镜（JWST）的高精度光谱数据将是验证这一点的关键。
