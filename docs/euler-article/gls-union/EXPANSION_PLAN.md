
## 五、写作风格指南

### 5.1 通俗化原则

每个概念必须包含：

1. **日常类比**：用生活中的例子
   - 例：统一时间刻度 = 三种不同品牌的手表对表

2. **视觉比喻**：用可视化场景
   - 例：边界 = 气球表面

3. **层层递进**：
   - 第1层：一句话概括
   - 第2层：日常比喻
   - 第3层：物理图像
   - 第4层：数学公式（可选）
   - 第5层：深入讨论

### 5.2 Mermaid图表规范

1. **引号包裹所有标签**：避免特殊字符问题
2. **不在图中使用LaTeX**：用文字描述数学关系
3. **颜色编码**：请自适应亮暗两种色调

### 5.3 公式呈现规范

关键公式使用"概念框"：

```markdown
**统一时间刻度母式**（全理论的DNA）
$$
\kappa(\omega) = \frac{\varphi'(\omega)}{\pi} = \rho_{\text{rel}}(\omega) = \frac{1}{2\pi}\text{tr}\,Q(\omega)
$$

**含义**：
- κ(ω)：统一时间密度
- φ'(ω)/π：散射相位导数
- ρ_rel(ω)：相对态密度
- tr Q/2π：群延迟平均

**一句话**：四种不同测量方法得到同一个时间刻度！
```

### 5.4 避免事项

❌ 不要：
- 大段公式堆砌
- 未经解释的术语
- 缺乏比喻的抽象论述
- Python代码（用户明确要求）
- 捏造不确定的内容

✅ 应该：
- 先比喻后公式
- 先直觉后严格
- 先整体后细节
- 读取源理论确保准确性
