**ζ值间接影响的情况**

$$\sum_n n^{-s} = \sum_{\{a_p\}} \prod_p p^{-s a_p} = \prod_p \sum_{a_p=0}^{\infty} (p^{-s})^{a_p} = \prod_p \frac{1}{1-p^{-s}}$$

**物理诠释**：
